# follower-party
Celebrate all the followers with a stylish post.

### Main features:
- Light / dark mode
- Add profile picture ( optional )
- Add name ( optional )
- @username ( optional )
- Add custom message 
- Add custom image to background
- Change background color
- Change font size
- Change font family
- Add bold to text
- Add italic to text 
- Option to select social media icon
